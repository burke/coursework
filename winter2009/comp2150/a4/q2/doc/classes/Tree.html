insert (data)

Insert an object into the subtree including and under this Node. First choose whether to insert into the left or right subtree, then either create a new node or insert into the existing node at the head of that subtree.

[show source] 
# File lib/tree.rb, line 21
  def insert(data)
    if !@data
      @data = data
    else
      node = (data.to_s < @data.to_s) ? :@left : :@right
      create_or_insert_node(node, data)
    end
  end

total ()

Sum all the numerical values in this tree. If this data object is a descendant of Numeric, add @data to the sum, then descend into both subtrees.

[show source] 
# File lib/tree.rb, line 32
  def total
    sum = 0
    sum += @data if (@data.is_a? Numeric)
    sum += [@left, @right].map{|e| e.total rescue 0}.inject(0){|a,v|a+v}
    sum
  end
